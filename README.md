# Retro Portrait MVP

Vintage portrait editor with cinematic presets and atmospheric film-inspired effects.

## Live Demo
https://retro-portrait-mvp.vercel.app

## Features
- Cinematic black & white presets
- Vintage newspaper effect
- Before / after comparison
- Responsive UI
- Dark minimal interface

## Stack
- React
- TypeScript
- Vite
- Vercel

## Status
MVP in active development.
